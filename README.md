## Quick Start

\*You need to set up [TailwindCSS configuration](https://www.sans-ui.org/docs/installation) first.

```bash
# npm
npm install @sans-ui/core

# yarn
yarn add @sans-ui/core

# pnpm
pnpm install @sans-ui/core
```
